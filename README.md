# 🚀 Devise – Premium Landing Page

Landing page premium desarrollada para **Devise**, enfocada en una experiencia visual fuerte, animaciones suaves y una estructura clara orientada a conversión.

Proyecto pensado para escalar a futuro (servicios, integraciones, pagos, etc.).

---

## 🧠 Tecnologías utilizadas

- ⚛️ **React** – Arquitectura de componentes
- ⚡ **Vite** – Build rápido y entorno de desarrollo ágil
- 🎨 **Tailwind CSS** – Estilos modernos y consistentes
- 🎥 **Framer Motion** – Animaciones fluidas y controladas
- 📩 **EmailJS** – Envío de formularios sin backend
- 🧩 **JavaScript (ES6+)**
- 🧱 **HTML5 / CSS3**

---

## ✨ Features principales

- 🎬 Video trailer al iniciar la web
- 🧭 Navegación fluida entre secciones
- 📱 Diseño **100% responsive**
- 📨 Formulario de contacto funcional
- 💎 Estética premium y minimalista
- ⚙️ Estructura preparada para futuras ampliaciones
